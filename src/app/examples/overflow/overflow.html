This demonstrates the use of .overflow-auto on an element. When the content exceeds the container's size, vertical scrolling is enabled automatically.

This example demonstrates the use of .overflow-hidden. Any content that exceeds the container's specified width or height will be clipped and not visible.

This example shows the use of .overflow-visible. When content exceeds the container's boundaries, it remains visible and is not clipped or hidden.

This example demonstrates the use of .overflow-scroll. When the content exceeds the width or height of the container, scrollbars are always visible, allowing users to scroll and view the hidden content.
