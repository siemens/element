<h1 align="center">@siemens/element-translate-cli</h1>

The Element Translate CLI is the command-line companion to the Element translation abstraction layer. It extracts translatable keys and default messages from compiled sources into TypeScript interfaces and JSON files. This keeps translations type-safe and in sync across mono-repo libraries.

**Get started, explore components and read the docs at [element.siemens.io](https://element.siemens.io).**
