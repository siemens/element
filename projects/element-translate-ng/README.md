<h1 align="center">@siemens/element-translate-ng</h1>

Element Translate is a translation abstraction layer for Angular. It lets you write framework-agnostic translation code that works with ngx-translate, Angular's built-in localize and other backends. This makes it easy to switch translation strategies without rewriting your application.

**Get started, explore components and read the docs at [element.siemens.io](https://element.siemens.io).**
