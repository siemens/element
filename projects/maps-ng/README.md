<h1 align="center">@siemens/maps-ng</h1>

Element Maps is an Angular mapping library based on OpenLayers. It provides themeable map components for displaying tiles, markers, clusters and overlays that match the Siemens Design Language. Use it to add interactive geospatial views to your applications.

**Get started, explore components and read the docs at [element.siemens.io](https://element.siemens.io).**
